## Run On Save


[Release of RunOnSave](https://github.com/qchen-fdii-cardc/runOnSave/releases/download/test/git-tracked-run-on-save-0.0.1.vsix)

Install the extension in VSCode by downloading the VSIX and then installing it:

```shell
curl -L -o git-tracked-run-on-save-0.0.1.vsix https://github.com/qchen-fdii-cardc/runOnSave/releases/download/test/git-tracked-run-on-save-0.0.1.vsix
code --install-extension ./git-tracked-run-on-save-0.0.1.vsix
```
