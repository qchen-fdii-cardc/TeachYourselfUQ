# 背景与动机

## 背景

不确定性是计量科学乃至科学中最重要的现代概念之一，而不确定度量化UQ则指的是对测量结果的可信度和准确性的评估。不确定性可能有人类的认知和信息模糊、对客体的无知造成，也可能因为不恰当的方法论和信仰系统造成，也会因为物理系统过程的随机本性而来。对于致力于观察和预测自然的科学研究而言，不确定性可能由数据引入，也可能因为方法引入，进一步分析可以认为其来源如图\ref{fig:uncertainty}所示分类，数据导致的不确定性包括自然不确定性、测量不确定性、参数不确定性和模型不确定性，而方法导致的不确定性包括辨识不确定性、建模不确定性、分析不确定性和表达不确定性。

![不确定性来源分类学\label{fig:uncertainty}](uncertainty.png)

量化不确定度，一直是科学研究中重点工作之一。而对于计量、试验鉴定和设备检验检测等三个领域，量化不确定性是这些相关学科的科学研究重要组成部分，同时也是三个领域及其交叉的工程应用基础支柱、工作深度的核心指标以及通向自主独立技术研发的桥梁。按照图\ref{fig:uncertainty}分析，这几个领域的研究工作都集中聚焦到物理过程的模型和不确定度的量化分析方法上。

随着科学研究和工程应用的复杂性增加，量化不确定性变得越来越重要。通过不确定度量化，我们可以更好地理解测量结果的可靠性，并为决策提供更有力的支持。这是计量工作的科学部分，也随着计量科学的发展，逐步走入计量工程实践。除计量本身，可计量性设计、全寿命计量保障等概念的提出和发展，都离不开对不确定性的深入理解和量化。

在作为试验鉴定和空气动力学探索的风洞试验中，不确定性同样扮演重要的角色。风洞试验是研究空气动力学特性的重要工具，但由于实验条件的复杂性和测量设备的限制，风洞试验结果往往伴随着一定程度的不确定性。因此，量化风洞试验中的不确定性对于提高试验结果的可靠性和指导后续研究具有重要意义。

设备安全性和可靠性的整个学科也依赖于对不确定性的深入理解和量化。可靠性分析和风险评估需要准确地量化不确定性，以对设备的性能和安全性进行评估，并指导设计和维护决策。从科学上和工程上，对不确定性量化进行深入研究是设备检验检测领域的一个重要方向。

## 动机

研究室在制定技术发展规划（特别是计量条令宣贯文件的制定和宣贯）过程中，经过反复论证和探索，大体明确了三个主体方向：

- 服务风洞试验鉴定的空白方向和急需方向开展计量研究，推动**全寿命计量保障**、**在线/原位计量**、**数字图像测量计量**、**动态、场量综合计量**等计量新思路、计量新技术的应用和发展，提升空气动力学计量的科学性和系统性，为风洞试验鉴定提供更有力的计量支撑
- 围绕风洞开展计量研究，提升风洞设计的系统性和科学性，为自主、独立、新型风洞设计提供**风洞可计量性设计**服务，推动风洞自主独立设计，为无人区的探索提供更有力的支撑
- 检测检验和计量的交叉方向，将计量学的科学方法和理论应用于**检测检验**领域，提升检测检验的科学性和准确性，深度开发和利用检测检验领域有前途的先进技术（如超声阵列检测技术、机器人自动化技术）

这三个方向的工作都离不开对不确定性的深入理解和量化，因此，开展不确定性量化的自主研究具有重要的意义和潜在的收益。
当然，受限于我们目前的研究深度和发表情况（仅有2篇SCI论文、4篇核心期刊论文、2篇会议论文、1项专利），我们很难独立申请到自然基金的项目来支持我们开展更加基础的研究工作。同时我们结合过去课题的研究，还是进行了相当的探索与积累。仔细分析我们目前发表的情况，我们的研究工作已经涉及到了不确定性量化的多个方面，虽然还没有形成系统的研究成果，但已经具备了一定的基础和积累。以下是我们目前发表的相关论文情况：

1. SCI论文：主题为测压管道频率响应的不确定度研究，发表在Measurment Science and Technology，IF 3.4，第一作者联培生颜明慧，通讯作者陈钦
2. SCI论文：主题为高压球罐热点线应力不确定度研究，发表在Mathematics，IF 2.2，第一作者西南科大联合指导学生吴泽鸿，通讯作者陈钦
3. 核心论文：主题为压力测量不确定度传播的Monte Carlo方法研究，发表在计量学报，第一作者联培生颜明慧，通讯作者陈钦
4. 核心论文：主题为超声阵列检测技术综述，发表在无损检测，第一作者刘春华，通讯作者陈钦
5. 核心论文：主题为超声阵列数据处理技术研究，发表在无损检测，第一作者联培生鲍子阳，通讯作者刘春华/陈钦
6. 核心论文：主题为高压球罐立柱垂直度检测机器人系统，发表在机械，第一作者西南科大联合指导学生宋伯霖，通讯作者刘春华
7. 会议论文：主题为风洞可计量性设计框架探索，发表在内部会议，第一作者赵少美，通讯作者陈钦
8. 会议论文：主题为风洞稳定段可计量性设计研究，发表在空气动力学大会，第一作者赵少美，通讯作者陈钦
9. 发明专利：主题为流热结构耦合快速计算算法，第一申请人陈鹏

我们的整个研究思路，试图用不确定度量化这一个核心概念，来串联起我们在计量、风洞设计和设备检验检测等三个领域的研究工作。目前，在一个技术基础课题、一个173课题的支持下，我们发展了GUM方法的不确定度评定、Monte Carlo方法的不确定度传播、NI-PCE方法的全局灵敏度分析等方面的研究成果。

此外，因为不确定度本身的科学特性，测量过程总是表达为一个多学科多物理的仿真模型，我们也不得不投入精力，围绕风洞、试验鉴定、检测检验来构造、验证和优化仿真模型，来支撑我们对不确定性的深入理解和量化。

为了推动和深化我们在多学科耦合模型与不确定性量化两方面的研究，最重要的就是争取能够开展基础研究的经费支持。在获得自然基金独立支持之前，我们还需要将现有研究工作的成果应用在不同的方面，寻求不同研究对象中的共性部分，来形成一个系统的研究体系，并通过发表高质量的论文来提升我们的研究影响力和竞争力。

目前最重要的一个方面就是173课题在耦合模型以及不确定度分析两方面的工作，有望能够在相关的领域获得基础研究的经费支持。目前，我们正在积极与李志辉团队（高速稀薄流动建模）、空间站团队、探月、火星探测团队等进行沟通，寻找合作机会，来推动我们在流声热耦合建模、不确定度分析等方面的研究成果的应用，一方面推动学术成果的固化（软著、专利和文章），另一方面是争取能够获得相关项目的经费支持。

# 必要性与可能的收益

科研工作，尤其是基础研究工作，往往需要较长的时间来积累和沉淀，才能够形成具有影响力的成果。持续性地投入经费和人力资源，对于基础研究的重要性毋庸置疑。研究室目前主体方向的基础研究工作已经取得了一些初步的成果，但要想在跨学科交叉领域形成系统的研究体系，还需要进一步持续投入，继续深入研究，为多物理场耦合建模和不确定性量化寻找更多的应用场景，争取各个方向的经费来推动研究工作向更高水平发展。最重要的一点，基础研究最需要持续进行研究人员的培养和团队建设，人员团队的技术能力是真正的资产和财富，人的发展和进步，才是真正的进步。高水平的研究队伍才能以不变应万变，满足未来所需要的研究工作，独立自主完成胜战所需的工程应用工作。

持续发展和推动多物理场耦合建模及其不确定性量化基础研究，对于研究室短、中期的收益可能包括：

1. 提升计量保障中特点问题、难点问题的解决能力，推动全寿命计量保障、在线/原位计量、数字图像测量计量、动态、场量综合计量等计量新思路、计量新技术的应用和发展，提升空气动力学计量的科学性和系统性，为空气动力计量技术基础的项目论证提供支持
2. 强化压力容器、结构加工制造产品的技术检测检验，推动检测、检验、监测数据的解读、解释和应用，提高新购检测检验设备的应用效率，支持对压力容器的全寿命科学管理，提升压力容器的安全性和可靠性
3. 建立风洞校准装置场量的多学科模型和不确定度分析技术能力，提升风洞校准装置的利用效率，推动场量的综合计量和量值传递，助力相关标准、规程的制订和完善
4. 积累非常规风洞及其核心部段的科学设计方法和技术，推动风洞自主独立设计，为无人区的探索提供更有力的支撑

考虑中、长期的发展，持续发展和推动多物理场耦合建模及其不确定性量化基础研究，对于研究室的收益可能包括：

1. 形成具有影响力的研究成果，提升研究室在多物理场耦合建模和不确定性量化领域的学术地位和声誉，提升研究室在机构、资质等方面的话语权，增强研究室的吸引力和竞争力，提高对优秀人才的吸引力
2. 培养储备具有多学科交叉背景的高水平研究人员，提升研究室的研究能力和创新能力，适应未来空气动力领域的不确定性挑战

# 原则与方案

## 原则

在规划接下来的基础研究方向和工作内容时，我们可以遵循以下几个原则：

1. **本立道生**，注重对基本概念、基本原理和基本方法的研究，确定多物理场耦合建模和不确定性量化作为基础研究工作方向，就是要追求实际工作与业务工作的最大公约数、抓住能够推动业务工作的技术和应用核心科学问题，来形成系统研究体系，推动在计量、风洞设计、试验鉴定和设备检验检测等领域的研究工作向更高水平发展，同时也还需要注重时刻反省和总结，目前的认识是否真正触及根本，是否真正把握了核心，而评价的标准就是是否能够产生成果，是否能够推动基础研究进步，如果不能就说明认识还不够深入，还没有真正把握核心问题，实践是检验整理的唯一标准
2. **君子不器**，理论和方法的研究不能被理论、方法本身所限制，局限于某一个特定的应用领域、局限于一条特定的技术路线、局限于某一类特定的问题，都是格局不够，要追求方法的基础性、普适性和通用性，来满足不同领域的研究需求，特别是在我们制定第一份技术发展规划才过去5年，我们的研究工作还处于积累和沉淀的阶段，理论和方法的研究更是如此，因此我们适当减少约束，保持开发的研究思路和多样性，一方面增加经费的申请机会，另一方面也能够避免基础研究工作过早地被应用领域的具体问题所限制，来保持研究工作的持续性和创新性
3. **躬身入局**，基础研究工作虽然是基础的，但也不能高高在上、空中楼阁，需要与应用领域紧密结合，推动研究成果的应用和转化，提升研究工作的影响力和价值，必然要求积极寻找应用场景，推动在多物理场耦合建模和不确定性量化方面的研究成果的应用，从不同的场景、不同的领域来验证和优化理论与方法，引入不同的视角和思路，尤其是不能用外协替代研究、用引进代替培养，研究人员必须要把手弄脏、深入实际、解决问题，身体力行、改造头脑，才能够真正推动研究工作向更高水平发展

## 方案

### 计划

在下一个阶段的工作中，考虑具体执行以下几个方面的计划：

1. 深度介入北研中心的相关气动融合轨道等方面的项目，引入玻尔兹曼统一方法，发展适用于计量、检测、风洞设计与试验鉴定领域的多物理场耦合建模技术，推动和考察不确定度量化方法在这些领域的应用，落实相关的经费支持，用以支持陈钦、陈鹏等研究人员的基础研究工作
2. 深入论证校准风洞场量的量值溯源体系，从现有的多孔探针、热线风速仪、噪声测量、压力脉动测量入手，积累风洞试验段动态量、场量测量过程的多物理场耦合模型和不确定度分析技术能力，推动风洞校准装置的利用效率，推动场量的综合计量和量值传递，助力相关标准、规程的制订和完善，这个方面的工作初步以技术基础课题的形式来支持，后续争取能够获得自然基金的项目支持，成果的产出争取以空气动力计量委员会的支持来推动，支持黄志阳、赵少美、苏欣、陈钦等研究人员的基础研究工作
3. 围绕压力容器、结构加工制造产品的技术检测检验，积累检测、检验、监测过程、仪器设备的多物理场仿真模型，通过不确定度量化的方法推动数据的解读、解释和应用，这个方面的工作初步以联培研究生和合作培养研究生来推进，后续争取能够获得自然基金的项目支持，支持刘春华、张国豪、陈钦等研究人员的基础研究工作
4. 逐步设计与实现风洞设计的可计量性设计框架，推动风洞自主独立设计，为无人区的探索提供更有力的支撑，这个方面的工作初步以计量技术基础的课题来间接支持，后续考虑是否能够引入足够的人力资源投入，目前尽在较低限度上投入赵少美、苏欣、陈钦等人员

### 预算

在预算方面，考虑到基础研究工作的持续性和长期性，必须要有一个合理的年度预算规划来支持研究人员的工作和研究活动，并且要根据研究工作的进展和需求进行适当的调整。在具体落实的项目间接支持之外，在能够申请到自然基金直接经费支持之前，基础研究大约每年需要大约15.5万元来开展活动（如图\ref{fig:budget}所示），需要研究室从不同渠道进行协调和筹集。

1. 参与相关领域的学术会议，提升研究人员的学术交流和合作机会，促进研究工作的进展和影响力，如测控专委会、中国计量学会年会、中国空气动力学大会等，预计每年参加3人次，每次费用约为0.6~0.8万元，总计约为2.0万元
2. 发表与项目没有直接关联的论文，提升研究人员的学术影响力和竞争力，促进研究工作的进展和认可，预计每年发表3篇核心论文，每篇费用约为0.6~0.8万元，总计平约为2.0万元；预计每年发表2篇SCI论文，每篇费用约为2.25万元，总计约为4.5万元，两项共计6.5万元
3. 出差参与相关领域和交叉领域的联合申请研讨、前期技术工作、申请书撰写和申报，初步规划差旅次数10次，每次平均3天，每次费用约为0.6万元，总计约为6.0万元
4. 参加1次交叉领域的暑期班、培训班等，提升研究人员的多学科交叉能力和创新能力，预计每次费用约为1.0万元，总计约为1.0万元

# 结语

基础研究的计划和经费支持关系到长远的技术储备和人才培养，也可能会对日常工作、业务工作的开展和执行产生一定的影响，需要领导的关心、支持和理解，来确保基础研究工作的顺利进行和持续发展。在执行过程中，也需要定期总结研究工作得失、反馈规划执行过程控制、评估规划制定科学性，更需要定期汇报研究工作进展和成果，展示基础研究工作价值和意义，争取更好的支持和资源投入。
